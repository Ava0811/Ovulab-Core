# Ovulab

Ovulab is an AI-assisted ovulation wellness platform designed to support women experiencing menstrual irregularities and PCOS-related symptoms through personalized nutrition and lifestyle guidance.

The platform combines cycle pattern analysis, wellness-focused recommendation systems, and conversational AI to generate personalized meal plans, lifestyle recommendations, and ovulation wellness support.

## Features

* Personalized ovulation wellness analysis
* AI-powered nutrition and lifestyle recommendations
* PCOS-friendly meal planning
* Calorie-aware dietary guidance
* Allergy and intolerance support
* Biomineral and vitamin recommendations
* Conversational AI wellness assistant
* Downloadable wellness reports
* Professional wellness dashboard UI

## Tech Stack

* Python
* Streamlit
* OpenRouter API
* DeepSeek LLM
* NumPy
* ReportLab

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
